IMPORTANT: Add your Christmas music file here!

1. Download any royalty-free Christmas music in MP3 or OGG format
2. Rename the file to: christmas_music.mp3 (or christmas_music.ogg)
3. Place it in this folder (app/src/main/res/raw/)

Recommended free sources:
- https://freemusicarchive.org/ (search for "Christmas")
- https://incompetech.com/music/ (Kevin MacLeod's royalty-free music)
- https://www.bensound.com/ (search for holiday music)

File naming rules:
- Use lowercase letters only
- No spaces or special characters
- Example: christmas_music.mp3

The app will automatically play this music when opened!
